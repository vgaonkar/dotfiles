# 3-step

## Purpose
A three-step AI prompt methodology for structured project planning: (1) expert anchoring to establish domain expertise, (2) interview-driven context gathering, and (3) RICECO-formatted execution prompt generation.

## Key Files

| File | Description |
|------|-------------|
| `step1-expert-anchor.md` | Step 1: Prompt to anchor the AI as a domain expert |
| `step2-interview-context.md` | Step 2: Interview prompt for gathering project requirements and constraints |
| `step2-master-guide.md` | Step 2 alternative: comprehensive master guide for context interviews |
| `step3-riceco-prompt.md` | Step 3: Generate a RICECO-formatted execution prompt from gathered context |
| `progress-tracker.md` | Template for tracking progress through the 3-step process |

## For AI Agents

### Working In This Directory

- These are **reference documents** describing a prompt engineering methodology.
- Steps are sequential: Step 1 sets expertise, Step 2 gathers context, Step 3 produces an actionable prompt.
- RICECO = Role, Instructions, Context, Examples, Constraints, Output format.
- Do not modify unless updating the methodology itself.
