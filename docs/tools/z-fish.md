# z (Fish plugin)

`jethrokuan/z` is a Fish-only directory jumper.

Upstream:
- https://github.com/jethrokuan/z

Notes
- Not wired by default in this repo.
- Overlaps with `zoxide`.

If you wire it into these dotfiles
- Add install/enable notes and any aliases in `home/dot_config/fish/config.fish.tmpl`.

Core usage
```fish
z proj
z -l
z -r
```
