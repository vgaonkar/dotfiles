# fzf.fish (Fish plugin)

`PatrickF1/fzf.fish` adds rich Fish keybindings and functions powered by `fzf`.

Upstream:
- https://github.com/PatrickF1/fzf.fish

Notes
- Not wired by default in this repo.
- For keybindings and conflicts, see `fish-keybindings-and-fzf.md`.

If you enable this plugin
- Decide whether you want to keep the repo's built-in Fish integration (`fzf --fish | source` in `home/dot_config/fish/config.fish.tmpl`).
- Avoid double-enabling unless you intentionally manage keybinding overlap.
