### Working In This Directory

- These are **point-in-time audit reports** -- they document the state of the project when the audit was run.
- Do not modify audit reports to reflect current state; instead, run a new audit.
- `FINAL-VERDICT.md` is the executive summary -- read it first for overall project health.
- `progress.json` tracks which audit dimensions have been completed.
