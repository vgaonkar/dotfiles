# Screenshots

Terminal screenshots for the dotfiles README.

## SVG Screenshots (fallback)

Generated by `generate-svg.py` — fake terminal renders using Catppuccin Macchiato colors.

```bash
python3 screenshots/generate-svg.py
```

## GIF Screenshots (VHS)

Install [VHS](https://github.com/charmbracelet/vhs) for real terminal recordings:

```bash
brew install vhs
cd screenshots/tapes
for tape in *.tape; do vhs "$tape"; done
```

Tape files are in `tapes/`. Each produces a `.gif` in the `screenshots/` directory.
