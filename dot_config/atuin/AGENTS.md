# atuin

## Purpose
Configuration for [Atuin](https://atuin.sh/), a shell history manager that replaces the default history with a searchable, synced database. This is a static config (not templated).

## Key Files

| File | Description |
|------|-------------|
| `config.toml` | Atuin settings: search mode, sync behavior, UI preferences, filter patterns |

## For AI Agents

### Working In This Directory

- This is a **static file** (no `.tmpl` suffix) -- identical across all platforms.
- Atuin config uses TOML format. See [Atuin docs](https://atuin.sh/docs/config/) for available options.
- Changes here affect shell history search behavior across all shells (Fish, Zsh, Bash).
